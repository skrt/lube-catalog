## 🔴 P0 — 横断的なトークン完成度のムラ（往復の主因・最優先）

### 1. フォーム共通トークンの欠落（input/select で入れたものが他に無い）
helperText 色/サイズ・errorText 色・required `*` 色・placeholder 色・radius が未定義:
- **text-area** — State カテゴリごと欠落（border/bg/text すべて無し）
- **combobox** — トリガー行トークン一式 + form-common 欠落
- **date-picker** — State カテゴリ欠落 + form-common 欠落
- **date-range-picker** — State カテゴリ欠落 + form-common 欠落
- **number-input** — 「Input との差分」記述のみで自己完結しない（base トークン全欠落）

### 2. トリガー箱の State トークン欠落（select で入れたものが他に無い）
border/bg を state 別に未定義: **combobox / date-picker / date-range-picker**

### 3. arbitrary pixel 値（no-fixed-pixels ルール違反 → 要確認）
- `w-[400px]`: **card**, **group-box**
- `w-[480px]`: **file-input**
- `w-[352px]`: **place**
- `min-w-[960px]`: **quote-table**
- `min-h-[1086px]`: **document**（A4 制約。等価クラス無し→報告対象）
- `size-[108px]/[220px]/[720px]`: **image**

### 4. Button のトークン不正値
- `Disabled` の `text-base-content/30)` — 末尾に余分な `)`（※ input/select と同じ系統のタイポ）
- Primary/Danger の hover が自己参照 `hover:bg-primary` / `hover:bg-error` → 正しくは `bg-primary-hover` / `bg-error-hover`（Secondary/Tertiary と記法不一致）

### 5. トークン/PROPS の構造欠落
- **table** — `props` 配列が無い（size 等の prop 未定義）
- **document** — `props` 配列が無い
- **icon-button** — Color トークンが無い（Common/Size のみ）
- **menu-button** — Active の text トークン `text-secondary-content` が描画 `text-primary` と不一致
- **header** — Container の背景/border-bottom トークン欠落
- **sidebar** — NavItem の text 色トークン欠落
- **filter-area** — Status Tabs が props/tokens 未定義
