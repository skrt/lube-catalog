Rules

  • トーンは「〜しましょう」推奨形に統一する。例外: duplicate / existence 系は状態説明なので「〜しています」「〜ません」も可。「〜できます」（許可形）は使わない
  • 表示位置は各フォームコンポーネントの errorText（text-xs text-error、フィールド下）。見た目のトークンは各コンポーネントの State / Common を参照
  • 文字数制限は固定値（32 / 256 / 512 / 5000）。実装側での t('errors.length.max', count: N) 補間運用との関係はエンジニアさんと要相談
  • 機能固有の文言は feature/{機能名}/{specifier} 命名で追加する（機能名は kebab-case）
  • i18n キーは暫定案。Rails config/locales/ja.yml の命名と擦り合わせて確定する
